# โจทย์ที่ 1: GET all products (1 คะแนน)

สร้าง endpoint สำหรับดึงรายการสินค้าทั้งหมด

**Path:** `GET /products`
 
**ไฟล์ที่ต้องสร้าง:**

- `src/products/product.module.ts` (class `ProductModule`)
- `src/products/product.controller.ts` (class `ProductController`)
- `src/products/product.service.ts` (class `ProductService`, method `findAll()`)
- `src/products/product.interface.ts` (interface `Product`)

**ไฟล์ที่ต้องแก้ไข:**

- `src/app.module.ts` (import และเพิ่ม `ProductModule`)

**เงื่อนไข:**

- อ่านข้อมูลจาก `data/products.json`
- Response ต้องเป็นรูปแบบ `ApiResponse<T>`
- message ต้องเป็น `"Fetched products successfully"`

**ตัวอย่าง Response:**

```json
{
  "success": true,
  "data": [
    {
      "id": 1,
      "name": "Laptop",
      "price": 25000,
      "stock": 10,
      "description": "High performance laptop"
    },
    {
      "id": 2,
      "name": "Mouse",
      "price": 500,
      "stock": 50,
      "description": "Wireless mouse"
    }
  ],
  "message": "Fetched products successfully"
}
```

**ข้อห้าม:**

1. ห้ามใช้ `process.exit()` มิเช่นนั้นจะตัดคะแนนเป็น 0 ทันที
2. ห้ามแก้ไข file `products.json` หรือ `purchases.json` หากใช้เพื่อทดสอบเสร็จแล้วให้แก้ไขเป็นข้อมูลเดิมหรืออย่า push ขึ้น git repository.
